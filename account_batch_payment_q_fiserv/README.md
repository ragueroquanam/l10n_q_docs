**Odoo Version:** 18.0

This module provides Fiserv specific batch payment file generation, extending the base batch payment functionality with Fiserv-compliant file formats and integration protocols.

## Features

* **Fiserv File Format Compliance**: Generates batch payment files in Fiserv's specific format requirements
* **Automated File Generation**: Automated generation of Fiserv-compliant payment files
* **Pre-configured Settings**: Pre-configured Fiserv-specific settings and validation rules
* **Validation and Error Checking**: Comprehensive validation system for format compliance

## Dependencies

* `account_batch_payment_q`: Base batch payment functionality
