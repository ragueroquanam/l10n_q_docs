**Odoo Version:** 18.0

This module provides POS linking functionality for the Quanam localization, enabling integration between POS systems and other business processes with data synchronization and workflow connectivity.

## Features

* **POS Integration**: Advanced integration between POS and other business modules
* **Data Synchronization**: Real-time data synchronization between POS and back-office systems
* **Workflow Connectivity**: Seamless workflow connections between POS operations and business processes
* **Transaction Linking**: Advanced transaction linking and cross-system reference management

## Dependencies

* `base`: Core Odoo functionality
