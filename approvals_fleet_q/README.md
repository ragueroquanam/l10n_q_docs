This module provides fleet-specific approval workflows for the Quanam localization, extending approval processes with fleet management integration, vehicle request approvals, and fleet operation authorization workflows.

## Features

* **Fleet Approval Workflows**: Specialized approval workflows for fleet operations and requests
* **Vehicle Request Management**: Comprehensive vehicle request and approval processing
* **Fleet Operation Authorization**: Advanced authorization controls for fleet-related operations
* **Integrated Fleet Management**: Deep integration with fleet management and approval systems

## Dependencies

* `base`: Core Odoo functionality
