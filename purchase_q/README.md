This module provides enhanced purchase order management for the Quanam localization, extending Odoo's standard purchase functionality with advanced workflow controls, supplier information management, and localized purchase processes.

## Features

### Enhanced Purchase Order Workflow

Advanced purchase order processing with localized workflow controls, custom approval processes, and enhanced document management tailored for regional business requirements.

### Supplier Information Management

Comprehensive supplier information management with enhanced vendor catalog integration, supplier performance tracking, and advanced supplier relationship management tools.

### Localized Purchase Processes

Purchase processes adapted to local business practices and regulatory requirements, including custom document types, tax handling, and compliance features.

### Advanced Reporting

Enhanced reporting capabilities for purchase analysis, supplier performance evaluation, and procurement insights with customizable dashboards and metrics.

## Configuration

### Purchase Settings

1. Go to *Purchase* > *Configuration* > *Settings*
2. Configure localized purchase parameters
3. Set up default values for purchase orders
4. Enable enhanced workflow features

### Supplier Configuration

1. Go to *Purchase* > *Suppliers*
2. Configure supplier information and catalogs
3. Set up supplier-specific terms and conditions
4. Configure supplier evaluation criteria

### Document Types

1. Configure localized document types for purchase orders
2. Set up custom numbering sequences
3. Define document validation rules
4. Configure approval workflows

## Usage

### Creating Purchase Orders

1. Go to *Purchase* > *Purchase Orders*
2. Create new purchase orders with enhanced workflow
3. Select suppliers and products from enhanced catalogs
4. Apply localized business rules and validations

### Supplier Management

1. Access *Purchase* > *Suppliers*
2. Manage comprehensive supplier information
3. Track supplier performance and history
4. Maintain supplier catalogs and price lists

### Purchase Analysis

1. Use enhanced reporting tools for purchase analysis
2. Monitor supplier performance metrics
3. Analyze procurement patterns and costs
4. Generate compliance and audit reports

## Technical Details

### Models

* Enhanced `purchase.order`: Extended purchase order functionality
