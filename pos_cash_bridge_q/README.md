**Odoo Version:** 18.0

This module provides a bridge between POS and cash management systems, facilitating seamless communication and data exchange between point-of-sale operations and cash handling processes.

## Features

* **POS-Cash Bridge**: Advanced bridge functionality between POS and cash systems
* **Data Exchange**: Seamless data exchange and communication protocols
* **Transaction Bridging**: Advanced transaction bridging between POS and cash operations
* **Integrated Cash Handling**: Unified cash handling across POS and cash management systems

## Dependencies

* `base`: Core Odoo functionality
