**Version:** 18.0.1.0.0  
**Odoo Version:** 18.0

This module provides functionality to force specific accounts in accounting moves, allowing override of default account selection for special accounting scenarios and manual account control.

## Features

* **Forced Account Selection**: Override default account selection in accounting moves
* **Manual Account Control**: Ability to manually specify accounts for special scenarios
* **Enhanced Move Processing**: Extended accounting move functionality with forced account capabilities
* **Flexible Account Assignment**: Support for custom account assignment rules and exceptions

## Dependencies

* `account`: Core accounting functionality
