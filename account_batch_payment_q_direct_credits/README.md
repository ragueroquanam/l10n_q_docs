**Odoo Version:** 18.0

This module provides Direct Credits specific batch payment file generation, extending the base batch payment functionality with direct credit transfer file formats and processing capabilities.

## Features

* **Direct Credits File Format**: Generates batch payment files for direct credit transfers
* **Automated File Generation**: Automated generation of direct credit payment files
* **Pre-configured Settings**: Pre-configured direct credit settings and validation rules
* **Validation and Error Checking**: Comprehensive validation system for format compliance

## Dependencies

* `account_batch_payment_q`: Base batch payment functionality
