**Odoo Version:** 18.0

This module provides purchase request functionality for the Quanam localization, enabling structured purchase request workflows, approval processes, and integration with purchase order generation.

## Features

* **Purchase Request Management**: Comprehensive purchase request creation and tracking
* **Approval Workflows**: Built-in approval processes for purchase requests
* **Purchase Order Integration**: Seamless conversion of approved requests to purchase orders
* **Request Tracking**: Complete tracking of request status and approval history

## Dependencies

* `purchase`: Core purchase functionality
